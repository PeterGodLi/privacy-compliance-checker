# 快速开始指南

## 🎯 5分钟快速上手

### 方式一：无需LLM，立即使用

1. **打开工具**
   - 直接在浏览器中打开 `index.html`

2. **输入应用信息**
   ```
   应用名称: 微信
   隐私条款URL: (可选)
   ```

3. **上传配置文件**
   - 选择对应平台的配置文件
   - 或点击"快速测试"查看演示

4. **查看结果**
   - 系统使用规则引擎自动分析
   - 生成详细的信息收集清单

### 方式二：使用LLM智能分析（推荐）

#### 第一步：配置LLM

**选项A：使用OpenAI（国际）**
```bash
1. 访问 https://platform.openai.com/
2. 注册账号并获取API密钥
3. 在工具中点击"LLM配置"
4. 选择"OpenAI"并输入API密钥
5. 保存并测试连接
```

**选项B：使用智谱AI（国内推荐）**
```bash
1. 访问 https://open.bigmodel.cn/
2. 注册账号并获取API Key
3. 在工具中点击"LLM配置"
4. 选择"智谱AI"并输入API密钥
5. 保存并测试连接
```

**选项C：使用Azure OpenAI（企业）**
```bash
1. 确保有Azure订阅
2. 创建Azure OpenAI资源
3. 部署模型（如gpt-4）
4. 在工具中配置API密钥和端点
5. 保存并测试连接
```

#### 第二步：开始分析

1. **返回主页面**
2. **输入应用信息和上传文件**
3. **点击"开始分析"**
4. **系统自动使用LLM进行深度分析**

## 💡 使用技巧

### 1. 快速测试
- 点击"快速测试"按钮可以立即看到演示效果
- 无需准备任何文件

### 2. 批量分析
- 可以同时上传多个配置文件
- 系统会自动识别平台并分别分析

### 3. LLM vs 规则引擎

| 特性 | LLM分析 | 规则引擎 |
|------|---------|---------|
| 准确度 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ |
| 速度 | 5-15秒 | <1秒 |
| 成本 | 需要API密钥 | 免费 |
| 适用场景 | 复杂隐私条款 | 标准配置文件 |

### 4. 节省LLM成本
- 先用规则引擎快速检查
- 对疑问项再用LLM详细分析
- 选择合适的模型（GPT-3.5比GPT-4便宜10倍）

## ❓ 常见问题速查

### Q: 没有API密钥能用吗？
**A:** 完全可以！系统会自动使用规则引擎。

### Q: 分析需要多长时间？
**A:** 
- 规则引擎：1秒内
- LLM分析：5-15秒

### Q: 数据安全吗？
**A:** 
- 所有数据仅在浏览器本地处理
- API密钥仅存储在本地localStorage
- 不会上传到任何第三方服务器

### Q: 支持哪些平台？
**A:** iOS、macOS、Android、Windows、鸿蒙OS

### Q: 如何获取更准确的结果？
**A:**
1. 使用LLM分析
2. 提供完整的隐私条款文本
3. 上传准确的配置文件
