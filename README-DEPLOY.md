# 部署说明

## 项目信息
- 项目名称：多平台应用隐私合规检查工具
- 项目类型：静态网站
- 技术栈：HTML + CSS + JavaScript

## 部署平台

### Vercel 部署（推荐）
1. 访问 https://vercel.com
2. 拖拽项目文件夹到部署区域
3. 等待部署完成
4. 获得链接：https://your-project.vercel.app

### Netlify 部署
1. 访问 https://netlify.com
2. 拖拽项目文件夹到部署区域
3. 等待部署完成
4. 获得链接：https://your-project.netlify.app

## 部署后检查
- [ ] 主页正常加载
- [ ] 平台按钮功能正常
- [ ] 文件上传功能正常
- [ ] LLM配置页面可访问
- [ ] 移动端显示正常

## 自定义域名
可在部署平台控制台绑定自定义域名。
